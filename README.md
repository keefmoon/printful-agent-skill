# Printful AgentSkill

This repository provides an [AgentSkill](https://agentskills.io/) designed to allow AI agents to interact seamlessly with the Printful REST API. 

By providing this skill directory to a compatible AI Agent, it immediately gains the context and instructional material necessary to manage a Printful store, browse the product catalog, create fulfillment orders, and generate product mockups.

## Features
- **Comprehensive API Guidelines**: `SKILL.md` natively includes an overview of Printful's authentication mechanisms, rate-limiting, and best practices.
- **Detailed Component References**: The `references/` directory contains 26 autogenerated, detailed markdown references for every specific part of the Printful API (Orders, Catalog, Mockups, etc.). This fully resolves the complex Printful OpenAPI specification, making it cost-effective and lightning-fast for an LLM to read.
- **Original API Specifications**: The original Printful OpenAPI `openapi.json` spec and a Postman collection are included for deep-dives into edge cases.

## Usage
1. Provide this directory to your AgentSkills-compatible AI agent.
2. Ensure you have obtained a Printful Private API Token or set up Public App OAuth credentials via the [Printful Developer Portal](https://developers.printful.com/).
3. Instruct the agent to perform tasks! (e.g., *"Use the Printful API to find the variant ID for a white unisex staple t-shirt, and create a draft order sending it to Sydney."*)

## Directory Structure
- `SKILL.md`: The core metadata and instructions for the agent.
- `references/`: Contains the full OpenAPI spec, Postman collections, and specific Markdown guides for each part of the API.
